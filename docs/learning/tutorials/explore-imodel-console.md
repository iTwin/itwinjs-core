# Using iModel Console

- Navigate to <https://imodelconsole.bentley.com/>
- Follow tutorial to learn how to use iModel Console
- Select your Project
- Select your iModel
- Choose most recent changeset

Execute queries
> Example: SELECT * FROM BisCore.Element
