What is iTwin.js?

iTwin.js is a library for creating, accessing, leveraging and integrating infrastructure digital twins. It was designed to be both flexible and open, so that it can be easily used and integrated with other systems.

  • Aligned

    Combine information from many sources with the iModel - CAD files, BIM files, databases, schematics, spreadsheets, etc.

  • Accountable

    Know who changes what in your digital twin but also when it has been made.

  • Accessible

    Make your your digital twin valuable for a multitude of use cases and an ulimited number of users by connecting it everywhere.

  • Approachable

    Minize your learning curve. iTwin.js is using the most modern and popular technologies and frameworks available.

  • Anywhere

    Write your application logic and user interface once and run it on Cloud, Web, Desktop, and Mobile devices.

  • Amazing

    Focus on what really matter. iTwin.js will handle the complexity of managing a large scale distributed digital twin efficiently.
