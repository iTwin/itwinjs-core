# FizzBuzz (nested IIF)

- dataset: AllProperties.bim

```sql
WITH RECURSIVE
  numbers (x) AS (
     SELECT 1 UNION ALL
     SELECT x + 1 FROM numbers
     LIMIT 15
  )
SELECT
  IIF(
    x % 3 = 0 AND x % 5 = 0, 'FizzBuzz',
    IIF(x % 3 = 0, 'Fizz',
      IIF(x % 5 = 0, 'Buzz', CAST(x AS VARCHAR))
    )
  ) AS [result]
FROM
  numbers
```
