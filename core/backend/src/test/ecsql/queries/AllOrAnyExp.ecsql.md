# Using ALL with recursive CTE

- dataset: AllProperties.bim

```sql
SELECT
  ECInstanceId,
  DirectLong
FROM
  aps.TestElement
WHERE
  DirectLong < ALL (
    WITH RECURSIVE
      cnt (x) AS (
        VALUES
          (1005)
        UNION ALL
        SELECT
          x + 1
        FROM
          cnt
        WHERE
          x < 1010
      )
    SELECT
      x
    FROM
      cnt
  )
```
