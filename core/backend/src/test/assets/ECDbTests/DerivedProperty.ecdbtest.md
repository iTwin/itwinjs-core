# Using Case

- dataset: AllProperties.bim

```sql
SELECT
  ECInstanceId,
  CASE
    WHEN DirectLong > (
      SELECT
        AVG(DirectLong)
      FROM
        aps.TestElement
    ) THEN 'Lower'
    ELSE 'Higher'
  END AS Column_Check
FROM
  aps.TestElement
```
